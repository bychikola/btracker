## 📋 Что нужно сделать для полного запуска

### 1. Настройте Supabase (если еще не сделали)

#### Шаг 1: Создайте проект
1. Перейдите на https://supabase.com
2. Войдите или зарегистрируйтесь
3. Нажмите "New Project"
4. Заполните:
   - Name: btracker
   - Database Password: (создайте надежный пароль)
   - Region: Europe West (или ближайший)
5. Нажмите "Create new project"
6. Подождите 2-3 минуты

#### Шаг 2: Получите API ключи
1. В левом меню: **Settings** → **API**
2. Скопируйте:
   - **Project URL**: `https://xxx.supabase.co`
   - **anon public** key: `eyJhbGciOiJIUzI1NiIs...`

#### Шаг 3: Обновите .env.local
Откройте файл `.env.local` и вставьте ваши ключи:

```bash
NEXT_PUBLIC_SUPABASE_URL=https://ваш-проект.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=ваш_anon_ключ
```

#### Шаг 4: Выполните SQL скрипт
1. В Supabase: **SQL Editor** → **New query**
2. Откройте файл `supabase/schema.sql` в проекте
3. Скопируйте весь SQL код
4. Вставьте в SQL Editor
5. Нажмите **Run** (или Ctrl+Enter)
6. Должно появиться: "Success. No rows returned"

#### Шаг 5: Отключите подтверждение email (для разработки)
1. В Supabase: **Authentication** → **Settings**
2. Найдите "Email Confirmations"
3. Отключите "Enable email confirmations"
4. Сохраните

#### Шаг 6: Перезапустите сервер
```bash
# Остановите текущий (Ctrl+C в терминале)
npm run dev
```

### 2. Протестируйте приложение

#### Тест 1: Регистрация
1. Откройте http://localhost:3000
2. Нажмите "Регистрация" в Header
3. Заполните форму:
   - Email: test@example.com
   - Username: testuser
   - Пароль: password123
   - Подтверждение: password123
4. Нажмите "Зарегистрироваться"
5. Вы должны автоматически войти

#### Тест 2: Проверка в Supabase
1. В Supabase: **Table Editor** → **profiles**
2. Вы должны увидеть нового пользователя

#### Тест 3: Избранное
1. На главной странице найдите любой матч
2. Нажмите на звездочку ⭐
3. Звездочка должна стать желтой
4. В Supabase: **Table Editor** → **favorite_matches**
5. Должна появиться запись

#### Тест 4: Профиль
1. Кликните на аватар в Header
2. Выберите "Профиль"
3. Должна открыться страница с вашими данными

#### Тест 5: Выход
1. Кликните на аватар
2. Выберите "Выйти"
3. Вы должны выйти из системы

## 🐛 Если что-то не работает

### Проблема: "Invalid API key"
**Решение**: 
- Проверьте `.env.local`
- Используйте `anon public` ключ (не `service_role`)
- Перезапустите сервер

### Проблема: "User already registered"
**Решение**: 
- Email уже используется
- Используйте другой email
- Или удалите пользователя в Supabase

### Проблема: Избранное не сохраняется
**Решение**:
- Убедитесь, что выполнили SQL скрипт
- Проверьте консоль браузера (F12)
- Проверьте, что авторизованы

### Проблема: "Email not confirmed"
**Решение**:
- Отключите подтверждение email в настройках Supabase
- Или проверьте почту и подтвердите email
