## 🎨 Как это выглядит

### Для неавторизованных пользователей
1. В Header отображаются кнопки "Войти" и "Регистрация"
2. Клик на "Войти" → открывается модальное окно входа
3. Клик на "Регистрация" → открывается модальное окно регистрации
4. Можно переключаться между окнами

### Для авторизованных пользователей
1. В Header отображается аватар (или первая буква username)
2. Клик на аватар → выпадающее меню:
   - Профиль (переход на /profile)
   - Выйти
3. Страница профиля показывает:
   - Аватар
   - Имя и фамилию (если указаны)
   - Username
   - Email
   - Дату регистрации

## 🔐 Безопасность

### Токены
- **Access Token** - хранится в localStorage, добавляется ко всем запросам
- **Refresh Token** - хранится в localStorage, используется для обновления access token

### Автоматическое обновление
- При получении 401 ошибки автоматически обновляется токен
- Если обновление не удалось → пользователь разлогинивается

### Защита маршрутов
- Страница `/profile` доступна только авторизованным
- Автоматический redirect на главную для неавторизованных

## 🚀 Как использовать

### 1. Регистрация нового пользователя

```
1. Откройте http://localhost:3000
2. Нажмите "Регистрация" в Header
3. Заполните форму:
   - Email: test@example.com
   - Username: testuser
   - Пароль: password123
   - Подтверждение пароля: password123
   - Имя: Тест (опционально)
   - Фамилия: Пользователь (опционально)
4. Нажмите "Зарегистрироваться"
```

### 2. Вход в систему

```
1. Нажмите "Войти" в Header
2. Введите:
   - Email: test@example.com
   - Пароль: password123
3. Нажмите "Войти"
```

### 3. Просмотр профиля

```
1. После входа кликните на аватар в Header
2. Выберите "Профиль"
3. Откроется страница с информацией о пользователе
```

### 4. Выход

```
1. Кликните на аватар в Header
2. Выберите "Выйти"
```

## 📝 Использование в коде

### Проверка авторизации

```tsx
import { useAuth } from '@/lib/contexts/auth-context'

function MyComponent() {
  const { isAuthenticated, user } = useAuth()

  if (!isAuthenticated) {
    return <div>Войдите в систему</div>
  }

  return <div>Привет, {user.username}!</div>
}
```

### Защищенная страница

```tsx
'use client'

import { useAuth } from '@/lib/contexts/auth-context'
import { useRouter } from 'next/navigation'
import { useEffect } from 'react'

export default function ProtectedPage() {
  const { isAuthenticated, isLoading } = useAuth()
  const router = useRouter()

  useEffect(() => {
    if (!isLoading && !isAuthenticated) {
      router.push('/')
    }
  }, [isAuthenticated, isLoading, router])

  if (isLoading) return <div>Загрузка...</div>
  if (!isAuthenticated) return null

  return <div>Защищенный контент</div>
}
```
